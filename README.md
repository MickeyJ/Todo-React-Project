##Todo App: React/ES2015 - Postgres/Knex - Express/Webpack
>
######you need to set up Postgres/Knex for Todo App functionality

```bash
  
$ npm install
  
```
>
######If you don't have webpack installed globally

```bash
  
$ npm install -g webpack
  
```
>
######Run webpack watch/build

```bash
  
$ npm run watch
  
```
>
######Run Express Server

```bash
  
$ npm start
  
     - or -
  
$ nodemon
  
```
